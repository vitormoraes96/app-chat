# app-chat
 ```
 Este é um Chat online simples onde é possivel conversar com varias pessoas.
 
 Foram utilizadas as tecnologias:
 
 NodeJS: onde foi criado o servidor, utilizando express juntamente com socket
 
 VueJS: Front end para construir o Chat com login
 
 Socket.IO: A melhor opção que encontrei no momento, no qual permite uma comunicação em tempo real,
 permitindo uma conexão de rapida resposta com servidor, ideal para um Chat!
 
 ```
## Processo de instalação
```
para instalar e acessar na localmente, efetue os seguintes passos:

~ clique no botão code e copie o caminho do repositorio

~ Vá no terminal e efetue "git clone https://github.com/vitormoraes96/app-chat.git"

~ Após clonar execute o comando "npm install"
(antes de executar o comando verifique se está na pasta correta")
```

### Executando o Projeto Localmente
```
Para executar digite no terminal "npm run serve"

após executar acesse o endereço localhost para acessar o app-chat
```
